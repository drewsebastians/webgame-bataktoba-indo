About

Tentang website ini

Batak Toba Play dibuat sebagai tempat belajar ringan untuk pemula: orang Batak diaspora, keluarga yang ingin mengenalkan bahasa ke anak, atau siapa pun yang ingin mengenal kosakata dasar Batak Toba - Indonesia.

Versi awal ini sengaja tidak mengejar seluruh corpus. Fokusnya adalah membuktikan dulu bahwa website nyaman dipakai, game playable, dan data subset cukup reliable untuk latihan awal.

Website berjalan full static di GitHub Pages. Tidak ada login, backend, database server, pembayaran, atau runtime AI.

Belajar kecil-kecil, tapi konsisten.
